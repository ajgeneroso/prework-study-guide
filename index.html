HTML

  • The head element contains inforemation about the webpage
  • The body element represents the visible content shown to the user
  • HTML elements work together to build the structure of a webpage and provide a framework that makes it easy for a user to navigate the webpage.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users
  • style element defines the CSS styles associated with a website
  • base element defines the base URL for a webpage
  • link element connects an external resource to the HTML document
  • meta element defines metadata such as the character set, description, keywords, author, and viewport
  • script element embeds client-side scripts in the HTML document

CSS

  • Margin property: indicates how much space we want around the outside of an element.
  • The padding property adds space around the content inside an element.
  • The img selector assigns property values to any image element in the markup.
  • The display property is assigned with the block value, which assigns each image to start on a new line
  • link element creates the connection to the external file.
  • ul elements declares the font
  • rel attribute specifies the relationship between the current document and the linked document resource
  • href attribute specifies the location (URL) of the external resource.

Git

  • git status: checks out what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: switches branches
  • git pull origin: pushes the current branch to the matching branch name in thr remote repository

JavaScript

  • JavaScript is the tool that takes a static HTML/CSS webpage and makes it interactive
  • Control flow is the order in which a computer executes code in a script
  • In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page
  • array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • function is simply a set of instructions that tells the computer how to perform a certain task.
